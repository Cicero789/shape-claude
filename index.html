Common questions

Do students need accounts?

No. Share a class link and they’re in. Teachers only.

Will it run on Chromebooks?

Yes — Shapely runs entirely in the browser using WebGL.

Can I customize the formulas shown?

You can switch between symbolic (a, b, h) and numeric labels with one click.

What standards does it support?

Common Core 6.G.1–4, 7.G.4–6, and most state equivalents.
